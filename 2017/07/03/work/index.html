work

步入社会的我们，一直在四面楚歌，四面碰壁。我们很迷茫，却又不知道如何朝着自己的目标前进。真的有这么难么？这几天就在向几位前辈请教，得来的结果还是体现自己的浮躁。过来人也许真的不能理解现在的90后怎么了。我们处在信息爆炸的时代，接受的信息过载，让我们始终不敢走出自己的惶恐。我们需要平静下来，想想自己真正的目标，破茧成蝶！这本书浅显易懂，直戳我的职业导向。从交互的设计流程告诉我如何慢慢的在项目中成长，如何才是专业的交互设计师。下面是我的读后小结，到底专业的交互设计师是如何工作的。
